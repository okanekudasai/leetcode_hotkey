솔브 커밋은:
- 리트코드의 문제 해결을 감지해서 코드를 깃허브에 푸쉬해 줍니다.
- 코드 실행을 위한 단축키를 제공합니다.
- 빠른 시일 내에 프로그래머스와 백준도 서비스 제공 예정입니다.
